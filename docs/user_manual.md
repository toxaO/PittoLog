# PittoLog 操作マニュアル

## 1. 基本

PittoLog は、バーコードを読み取って物品の貸出と返却を管理するアプリです。

主に使う画面は次の2つです。

- `貸出・返却`: 日常の貸出・返却作業
- `登録・設定`: 物品、カテゴリ、部署、バーコード、出力先などの管理

バーコードリーダーは、キーボード入力として動作するものを想定しています。

## 2. 貸出・返却

### スキャンを開始する

`貸出・返却` 画面を開き、左側のスキャンエリアをクリックします。
読み取り欄にフォーカスが当たると、バーコードをスキャンできます。

### 貸出する

1. 貸出する物品バーコードをスキャンします。
2. 貸出先の部署バーコードをスキャンします。
3. 内容を確認し、確認バーコードをスキャンします。

キャンセルしたい場合は、キャンセルバーコードをスキャンします。

### 返却する

1. 貸出中の物品バーコードをスキャンします。
2. 内容を確認し、確認バーコードをスキャンします。

キャンセルしたい場合は、キャンセルバーコードをスキャンします。

### 貸出中物品を確認する

右側の `貸出中物品` で、現在貸出中の物品を確認できます。

検索、カテゴリ、部署、並び順を指定できます。
CSV出力を押すと、現在の検索条件を反映したCSVを出力します。

## 3. 物品の登録と変更

`登録・設定` の `物品` タブで操作します。

### 物品を登録する

1. 先にカテゴリを登録します。
2. `物品登録` を押します。
3. 物品名とカテゴリを入力します。
4. 通常はバーコード番号を手動入力せず、自動採番を使います。

手動でバーコードを指定する場合だけ、手動指定を有効にします。

### 物品を変更する

1. 物品一覧から対象行を選択します。
2. `物品登録情報変更` を押します。
3. 物品名、カテゴリ、必要な場合のみバーコードを変更します。

### 物品を無効にする

物品一覧から対象行を選択し、`有効/無効切替` を押します。

通常の一覧では有効な物品だけを表示します。
無効な物品を確認したい場合は、`有効/無効` の絞り込みを変更します。

## 4. カテゴリ

`登録・設定` の `カテゴリ` タブで操作します。

- `カテゴリ登録`: 新しいカテゴリを追加
- `カテゴリ登録情報変更`: 選択したカテゴリ名を変更

物品登録時は、ここで登録したカテゴリを選択します。

## 5. 部署

`登録・設定` の `部署` タブで操作します。

- `部署登録`: 新しい部署を追加
- `部署登録情報変更`: 選択した部署名やバーコードを変更
- `登録解除`: 部署を使わない状態にする

部署バーコードは貸出先の指定に使います。

## 6. 履歴

`登録・設定` の `履歴` タブで、貸出と返却の履歴を確認できます。

検索、操作、部署、期間、並び順を指定できます。
CSV出力を押すと、現在の検索条件を反映したCSVを出力します。

## 7. バーコード発行

`登録・設定` の `バーコード発行` タブで、バーコードを作成できます。

### 1件ずつPNGを作成する

1. 種類を選びます。
   - `ITEM`: 物品
   - `DEPT`: 部署
   - `ACTION`: 確認、キャンセル
2. `ITEM` の場合はカテゴリを選びます。
3. 対象を選びます。
4. `PNG作成` を押します。

保存先は、設定タブの `PNG出力先フォルダ` を基準に開きます。

### A4まとめPDFを作成する

`印刷するバーコード` 欄に、次の形式で入力します。

```text
物品1,ITEM:000001
総務,DEPT:0001
確認,ACTION:001
キャンセル,ACTION:002
```

`A4まとめPDF作成` を押すと、A4形式のPDFを作成します。

物品一覧で物品を選択して `印刷リストへ追加` を押すと、選択した物品をこの一覧へ追加できます。

## 8. 設定

`登録・設定` の `設定` タブで、出力先と時間を設定します。

- CSV出力先フォルダ
- PNG出力先フォルダ
- PDF出力先フォルダ
- 読み取り中のリセット時間
- 読み取り結果を表示する時間

変更後は `設定を保存` を押してください。

## 9. CSV読み込み

物品はCSVからまとめて登録できます。

`物品` タブで `CSV読み込み` を押し、CSVファイルを選択します。

対応列:

- `物品名` または `name`: 必須
- `カテゴリ` または `category`: 任意
- `バーコード` または `barcode`: 任意

バーコードが空欄の場合は、自動採番されます。
CSV内に新しいカテゴリがある場合は、自動でカテゴリ登録されます。

## 10. 困ったとき

### スキャンしても反応しない

スキャンエリアをクリックし、読み取り欄にフォーカスを当ててください。

### 未登録バーコードと表示される

物品、部署、操作用バーコードが登録済みか確認してください。

### 貸出できない

物品がすでに貸出中の可能性があります。
貸出中物品一覧を確認してください。

### CSVやPDFの保存先を変えたい

`登録・設定` の `設定` タブで、出力先フォルダを変更してください。
